# 🚀 DeepHire AI

DeepHire AI is an intelligent recruitment platform that automates resume screening, ATS scoring, candidate ranking, AI-powered interviewing, and recruitment analytics.

The platform helps recruiters identify the best candidates faster using Artificial Intelligence, Natural Language Processing, Semantic Similarity, and Large Language Models.

---

## ✨ Features

### 📄 ATS Resume Analysis

* Resume parsing and extraction
* ATS compatibility scoring
* Skill gap analysis
* Resume recommendations

### 🤖 AI Recruitment Assistant

* AI-powered recruiter chatbot
* Candidate evaluation support
* Job description understanding

### 🎤 AI Interviewer

* Automatic interview question generation
* Technical and HR interview support
* Candidate assessment assistance

### 📊 Recruitment Analytics

* Candidate ranking dashboard
* ATS score visualization
* Skill distribution analysis
* Recruitment performance insights

### 📂 Candidate History

* Candidate database management
* Search and filtering
* CSV export support

### 👤 User Profile & Settings

* Personalized recruiter profiles
* Theme management
* Account statistics
